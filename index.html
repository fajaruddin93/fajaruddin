eAbsensi Mobile
Mobile

Mobile Application

Android, IOS Application

The PUPR eAbsensi Mobile application provides a solution for PUPR Employees to attend attendance electronically without having to queue in making absentees coming in and going home because these applications can be installed on any smartphone with the Android platform.

The Mobile Application utilizes the smartphone's camera feature to make attendance. This is intended to minimize the absence of attendance by employees so that it can improve the accuracy of employee absences.

Each employee can log into the application using the account of each employee. This application can check your location via GPS on your smartphone, so employees can only make attendance in the area that is allowed to do attendance, namely in the PUPR Office environment.

It is expected that this application can provide convenience in absenteeism for employees in the PUPR worker environment.
